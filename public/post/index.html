About

What this course is not

This is not an introductory statistics course. Running a stats test is the easy part (interpreting the results of that test is a whole other thing though). The hard part of any data analysis is getting the data into shape to be explored and analysed.

What this course is

This short-course provides an introduction to data wrangling, plotting, and reproducible research practices using the statistical computing programme R and the distributed version control system git. To facilitate the workflow of using R and git, we will use RStudio, an integrated development environment (IDE) for R that includes support for git. In addition, you will learn to use GitHub, a web-based git repository hosting service. GitHub is an essential tool for producing reproducible and open data analysis.

Preparing for the course

Do I need any experience using R or git?

The course assumes some basic knowledge of R and git. If the commands data[ , 1:5] or git commit mean absolutely nothing to you, then you need to complete one or more introductory online courses in preparation for this course (see below for some good free options), and the ‘R Programming’ course provided through the R package swirl (instructions below).

What software must I install on my computer?

You will need the latest releases of the following apps (if you already the apps installed, check for updates):

  • R (available for: Windows, Mac, and Linux)

  • RStudio Desktop (available for: Windows, Mac, and Linux) It is important that you only install RStudio after you have installed R.

  • git (available for: Windows, Mac, and Linux) After you have installed git, you will need to configure the app. You can download documentation on how to perform this configuration here.

  • Although it is not essential, I suggest you also install TeX/LaTeX onto your system if you are a Windows or Mac user. The size of the download is very large (~2GB).

Once you have downloaded and installed R and RStudio, you will need to install the following R packages (packages add extra functionality to the basic R installation):

  • knitr, RColorBrewer, rmarkdown, swirl, svglite, tidyverse 1, viridis

To install the packages from the R console in RStudio:

# Re-type or copy and paste the appropriate text into the R console 
# and press 'Enter'.

install.packages(c('knitr', 'RColorBrewer', 'rmarkdown', 
                   'swirl', 'svglite', 'tidyverse', 
                   'viridis'))

What services must I sign-up to?

You need to sign-up for a free GitHub account. And if you complete any of the free online courses in preparation for this course you will probably have to sign-up to the organsizations offering the courses.

Do I need to pay for any of these apps?

No. The reason for choosing these apps and services is that they are free (as in beer and as in speech) and work across all major operating systems.

Troubleshooting

If you have any issues installing the software, please contact me. The most common problem people encounter is getting R/RStudio to communicate through corporate proxy servers, such as those used by the University of the Witwatersrand (proxyad.wits.ac.za:80).

Swirl

The swirl package for R was developed by the Swirl Development Team, and includes a suite of step-by-step interactive training courses on R, which are aimed primarily at the novice and intermediate user. Before starting this course you should at the very least be familiar with the content of the swirl course: ‘R Programming’. During the course, we shall also complete all or parts of the swirl courses: ‘Getting and Cleaning Data’, and ‘Exploratory Data Analysis’. You can look at the full suite of swirl courses that are available here.

Follow the instructions below to install and load the swirl package:

# Re-type or copy and paste the text below into the R console, 
# pressing 'Enter' after each step.

# If you haven't already installed swirl
install.packages('swirl')

# Once the package is installed, load the 'swirl' package
library(swirl)

# Launch a 'swirl' session and follow the prompts
swirl()

…and to install swirl courses:

# Re-type or copy and paste the text below into the R console, 
# pressing 'Enter' after each step.

# Load the 'swirl' package
library(swirl)

# Download a course from the 'swirl' github repository
## Course names:
### 'R Programming Alt'
### 'Getting and Cleaning Data'
### 'Exploratory Data Analysis'
install_from_swirl('Course Name Here')

# Launch a 'swirl' session and follow the prompts
swirl()
